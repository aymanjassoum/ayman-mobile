# 📱 مركز أيمن لصيانة الموبايل

موقع إلكتروني موحد لإدارة مركز الصيانة وعرض حالة الأجهزة، مصمم باستخدام HTML وJavaScript ومربوط مباشرة بـ Google Sheets.

---

## 🔧 التبويبات المتوفرة

- 🔍 **حالة الجهاز**  
  إدخال الكود الرباعي لعرض حالة جهاز الزبون من Google Sheet

- ➕ **استلام جهاز جديد**  
  نموذج لتسجيل بيانات الجهاز، وتوليد كود رباعي وQR داخل وصل إلكتروني

---

## 🛠️ التقنية المستخدمة

- HTML / CSS / JavaScript
- Google Apps Script Web App
- Google Sheets API
- QR Code (مكتبة qrcode.min.js)
- شعار احترافي وهوية مرئية موحدة

---

## 📂 ملفات المشروع

| الملف                | الوصف                        |
|----------------------|-------------------------------|
| `index.html`         | الصفحة الموحدة بتبويبات ديناميكية |
| `style.css`          | التنسيقات العامة للواجهة       |
| `js/script.js`       | منطق التبويبات + توليد الكود   |
| `js/qrcode.min.js`   | مكتبة توليد رمز QR            |
| `assets/logo.png`    | شعار مركز الصيانة              |

---

## 🚀 خطوات الاستخدام

1. اربط Google Sheet بخدمة Apps Script
2. أنشئ Web App للقراءة وأرسل الرابط إلى `script.js`
3. فعّل GitHub Pages من إعدادات المستودع
4. شارك الرابط مع الزبائن
